# Smart Waste Management System

A web-based platform designed to optimize municipal waste collection using Machine Learning predictions and Route Optimization algorithms.

# features
Smart Prediction: Real-time fill level predictions using AI models based on historical data.
Route Optimization: Interactive map interface showing collection routes.
User Authentication: Role-based access for Admins and Residents.
Bin Management: Full CRUD interface for managing waste bin locations.

# Technologies Used
Frontend: HTML5, CSS3, Vanilla JavaScript, Leaflet.jsBackend / Core**: Python, Flask, Pandas
AI/ML: Scikit-Learn (Joblib)

# Responsibilities
- Documentation & Reporting: `README.md`, `REPORT.md`, ER Diagram creation.
- Data Management: Maintenance and structure of `routes.json`.


# How to Run locally
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python app.py`
4. Access the web interface at `http://127.0.0.1:5000`
